# HostIt-AI

HostIt-AI is a web platform designed to facilitate comprehensive event management through a centralized interface.  
The goal of the project is to allow users to organize, manage, and track events from a single place, integrating planning, communication, and e-commerce features.

## âš™ Installation and Setup

### Prerequisites

- Python 3.x  
- Node.js  
- npm or yarn  

---

### Backend Setup

1. Navigate to the backend folder:

cd backend-flask

2. Create a virtual environment (optional but recommended):

python -m venv venv
source venv/bin/activate   # Linux/Mac
venv\Scripts\activate      # Windows

3. Install dependencies:

pip install -r requirements.txt

4. Run the server:

python main.py

---

### Frontend Setup

1. Navigate to the frontend folder:

cd frontend-vue

2. Install dependencies:

npm install

3. Run the application:

npm run serve

The application will usually be available at:

http://localhost:8080
